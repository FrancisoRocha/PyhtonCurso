# Práctica Variables

## Objetivo

Crear un **Menú del Día de un Restaurante Gourmet** usando variables y la función `print` para mostrar los platos en la consola.

---

## Instrucciones

Escribe un programa en **Python** que utilice variables para almacenar información sobre diferentes platos y luego usa la función `print` para mostrar un menú gourmet.

El menú debe contener tres secciones:

- Entrante
- Plato Principal
- Postre

Para cada sección, define dos variables:

- Una para el nombre del plato:

  - `nombre_entrante`
  - `nombre_principal`
  - `nombre_postre`

- Otra para el precio:
  - `precio_entrante`
  - `precio_principal`
  - `precio_postre`

Usa múltiples llamadas a la función `print` para mostrar primero el nombre del plato y luego su precio en líneas separadas.

**Importante:**  
Las variables que definas para crear el menú deben tener exactamente los mismos nombres indicados anteriormente.

---
