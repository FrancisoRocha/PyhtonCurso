Práctica: Strings, Indexing, Slicing y Stride
Objetivo

Descifrar los “Mensajes Secretos de una Carta de Amor” utilizando strings, indexación, slicing, stride y strings multilínea en Python.

Instrucciones

Se te proporcionará una carta de amor que contiene tres párrafos, cada uno con un mensaje secreto oculto.

Tu tarea es escribir un programa en Python que utilice técnicas de indexación, slicing y stride para descifrar estos mensajes.

Cada párrafo tiene un patrón de codificación diferente:

Primer párrafo

El mensaje secreto está formado por cada decimoctavo (18) carácter

Comenzando desde el segundo carácter

Segundo párrafo

El mensaje secreto se encuentra entre el carácter 138 y 147 de la cadena de texto

Tercer párrafo

El mensaje secreto está formado por letras ubicadas en las siguientes posiciones de la cadena:

Primera letra: posición 125

Segunda letra: posición 94

Tercera letra: posición 35

Cuarta letra: posición 107

Quinta letra: posición 20

Sexta letra: posición 1

Requisitos

Usa print para mostrar los mensajes secretos decodificados.

Ten en cuenta que las posiciones indicadas empiezan en 0, es decir, el índice 0 corresponde al primer carácter.
