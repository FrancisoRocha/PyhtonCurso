# Práctica Sentencia return y Docstrings

## Objetivo
Desarrollar un "Generador de Mensajes Personalizados" utilizando funciones en Python. El enfoque principal será en la sentencia `return`, la implementación de argumentos, el uso de strings y la incorporación de docstrings para la documentación de funciones.

## Instrucciones
1. Escribe un programa en Python que contenga una función llamada `generar_mensaje` para crear mensajes personalizados.
2. Incluye un docstring al inicio de la función que describa su propósito, los parámetros que toma y lo que devuelve.
3. La función debe recibir dos argumentos:
   1. `nombre` (argumento posicional). El nombre de una persona.
   2. `mensaje` (parámetro con valor por defecto, "Bienvenido al curso de Python").
4. La función debe crear y devolver un mensaje completo utilizando la sentencia `return`, combinando el nombre y el mensaje predeterminado. Asegúrate de que el docstring refleje esta funcionalidad.
5. Después de definir la función, realiza una llamada a `generar_mensaje` con un nombre específico y muestra el resultado.

## Resultado esperado
```
"¡Hola, Santiago! Bienvenido al curso de Python"
```
